# Finding the Plugin's Settings

<figure><img src="../.gitbook/assets/First glance at the Pattern Wrangler Settings page.png" alt="Pattern Wrangler Admin"><figcaption><p>Pattern Wrangler Admin</p></figcaption></figure>

## Finding the Admin Settings From the Plugins Screen <a href="#finding-the-admin-settings-from-the-plugins-screen" id="finding-the-admin-settings-from-the-plugins-screen"></a>

<figure><img src="../.gitbook/assets/Locating pattern wrangler settings from the plugins page.png" alt="Pattern Wrangler on the Plugin&#x27;s Screen"><figcaption><p>Pattern Wrangler on the Plugin's Screen</p></figcaption></figure>

If you are on the plugin's screen, search for **Pattern Wrangler** and click on the `Settings` link.

### Finding the Admin Settings from the Menu

<figure><img src="../.gitbook/assets/Locating Pattern Settings From Admin Dashboard.png" alt="Patterns->Settings to Find the Pattern Wrangler Settings"><figcaption><p>Patterns->Settings to Find the Pattern Wrangler Settings</p></figcaption></figure>

Alternatively, in the admin menu, head to `Patterns->Settings`.
